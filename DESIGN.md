# Hairo — Дизайн-система

> Визуальный язык приложения. Источник: `Hairo Design.dc.html` (Claude Design, Фаза 0.5).
> Mobile-first, подхват светлой/тёмной темы Telegram Mini App, тексты переживают RU/ES/DE.
> Реализация — фронтенд Фазы 1 (Nuxt + Tailwind). Токены ниже мапятся в `tailwind.config` / CSS-переменные.

---

## 1. Бренд

- **Имя:** Hairo.
- **Логотип:** знак — мягкая дуга-локон (отсылка к причёске и зеркалу) + вордмарк в Cormorant. Есть варианты на светлом/тёмном/акцентном фоне и иконка-знак отдельно.
- **Эстетика:** «бьюти-редакция» — тёплый премиум без вычурности.
- **Тон голоса:**
  - ✓ Тёплый, на «вы», без официоза: «Загрузите селфи — покажем, что вам идёт».
  - ✓ Конкретно, без хайпа про «нейросети» — говорим о результате, не о технологии.
  - ✕ Не давим и не обещаем «идеал» — образ это примерка и вдохновение, не приговор.

## 2. Типографика

Google Fonts: **Cormorant** (serif, дисплей/заголовки) + **Manrope** (sans, UI/текст/кнопки).

## 3. Цветовые токены

### Светлая тема
```css
--bg:           #FAF7F2;  /* фон */
--surface:      #FFFFFF;  /* поверхность (карточки) */
--surface-2:    #F3EDE4;  /* поверхность-2 */
--accent-soft:  #F6E7DE;  /* акцент-фон (выбранное, бейджи) */
--accent:       #C26A4A;  /* акцент (CTA, активное) */
--accent-dark:  #A4502F;  /* акцент тёмный (метки, ссылки) */
--text:         #211C18;  /* текст */
--text-muted:   #6B6259;  /* текст тихий */
--border:       #ECE3D6;  /* границы */
--border-strong:#E2D7C9;  /* границы инпутов/кнопок */
--success:      #5B8C6E;
--danger:       #C0392B;
```

### Тёмная тема (Telegram dark)
```css
--bg:           #16130F;
--surface:      #211C17;
--surface-2:    #2C261F;
--accent-soft:  #46332A;  /* выбранная карточка в dark */
--accent:       #E08862;  /* акцент светлеет для контраста */
--text:         #F3ECE2;
--text-muted:   #A89C8D;
--border:       #2C261F;
--success:      #5B8C6E;
```

- **AI-акцент (градиент):** `linear-gradient(135deg,#C26A4A,#E08862)`.
- **Маппинг на Telegram `theme_params`:** `bg_color`→`--bg`, `text_color`→`--text`, `hint_color`→`--text-muted`, `button_color`→(системные кнопки). **Брендовый акцент держим собственный** — не подменяем кнопкой Telegram.

## 5. Компоненты

- **Кнопки:** primary (заливка `--accent`, h52, `--r-btn`, тень `--shadow-cta`); dark/CTA (`--text` фон, белый текст — для «Сгенерировать · N кредитов»); secondary (бордер 1.5px `--border-strong`, прозрачный фон); pill-text (фон `--surface-2`, `--r-pill`).
- **Поля ввода:** h52, `--r-input`, бордер 1.5px `--border-strong`.
- **Чипы (фильтры/повод):** `--r-pill`; выбранный — заливка `--accent`/`--text`, невыбранный — `--surface-2` или бордер.
- **Чекбокс:** 22px, радиус 7, заливка `--accent` + белая галка.
- **Прогресс визарда:** полоски h6, `--r-pill`; пройдено — `--accent`, осталось — `--border-strong`.
- **Карточка-опция (визард):** `--r-card`; выбранная — бордер 2px `--accent` + фон `--accent-soft`, кружок-галка.
- **Карточка образа:** `--r-card`, превью-картинка, бейдж совпадения (белый pill, % справа сверху), лайк-сердце (слева сверху), название в Cormorant, подзаголовок (длина/деталь) в `--text-muted`. Выбранная — бордер 2px `--accent`.
- **Нижняя панель действий** (каталог/результат): фон `--surface`, верхние углы `--r-sheet`, тень `--shadow-bar`.

## 7. Что синхронизировано в продукт

Дизайн внёс конкретику → перенесено в `PRODUCT.md`:
- Пользовательский термин для генерации — **«примерка»** (1 кредит = 1 примерка).
- Score фильтрации показываем как **% совпадения** на карточке.
- Реферал-бонусы — плейсхолдеры (до финмодели). Пакеты цен пересмотрены в Фазе 4 по COGS + Stripe-комиссии (5/€4,99 · 20/€14,99 · 50/€29,99, без пака «1 шт») — актуальные значения в `PRODUCT.md` §4.2 и `ROADMAP.md`.

## 8. Интеграция (Фаза 1)
