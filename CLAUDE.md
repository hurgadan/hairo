# Hairo

AI-сервис примерки причёсок: пользователь загружает селфи → визард вопросов → подбор из каталога причёсок → генерация фото с выбранной причёской. Web-first, с обёрткой в Telegram Mini App.

**Язык общения в проекте: русский.**

## Документы (читать перед работой по теме)

- **[PROJECT.md](PROJECT.md)** — бизнес, стратегия, рынок, дистрибуция, что уже решено.
- **[PRODUCT.md](PRODUCT.md)** — продукт: пользовательский флоу, визард вопросов.
- **[CATALOG.md](CATALOG.md)** — каталог причёсок: таксономия, схема карточки, логика фильтрации.
- **[PROMPTS.md](PROMPTS.md)** — промпт-шаблоны (улучшение, генерация, каталог).
- **[DESIGN.md](DESIGN.md)** — дизайн-система: бренд, токены (light/dark), компоненты, экраны.
- **[ROADMAP.md](ROADMAP.md)** — фазовый план разработки (вехи).
- **[ARCHITECTURE.md](ARCHITECTURE.md)** — инженерные правила бэкенда (обязательны к соблюдению).
- **[TECH.md](TECH.md)** — стек и открытые инженерные вопросы.

> Держать документы в актуальном состоянии — решения фиксировать в соответствующем файле, не смешивая слои.
> Для документации по NestJS, grammY, Stripe, TypeORM и др. библиотекам — использовать MCP `context7`, а не память модели.

## Кратко зафиксировано

- Качество модели валидировано вручную (Валерия, жена основателя).
- Платформа: **web-first**; Telegram Mini App = обёртка над тем же web-app. WhatsApp — только уведомления/ссылки.
- Канал: пан-европейская русскоязычная диаспора в Telegram (плацдарм) → коренной DACH. i18n RU/ES/DE.
- Биллинг: ИП (Einzelunternehmen, DE) + Stripe.
- Стек: **Node.js + NestJS** (backend), **Nuxt.js** (frontend). Модель генерации — Nano Banana / Gemini 2.5 Flash Image.
