# Hairo

AI-сервис примерки причёсок: пользователь загружает селфи → визард вопросов → подбор из каталога причёсок → генерация фото с выбранной причёской. Web-first, с обёрткой в Telegram Mini App.

**Язык общения в проекте: русский.**

## Кратко зафиксировано

- Качество модели валидировано вручную (Валерия, жена основателя).
- Платформа: **web-first**; Telegram Mini App = обёртка над тем же web-app. WhatsApp — только уведомления/ссылки.
- Канал: пан-европейская русскоязычная диаспора в Telegram (плацдарм) → коренной DACH. i18n RU/ES/DE.
- Биллинг: ИП (Einzelunternehmen, DE) + Stripe.
- Стек: **Node.js + NestJS** (backend), **Nuxt.js** (frontend). Модель генерации — Nano Banana / Gemini 2.5 Flash Image.
